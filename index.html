Projects

Cozey Cradles

Cozy Cradle is a baby car seat rental service that allows users to rent car seats for their infants. This web application is built using the MERN stack (MongoDB, Express, React, Node.js). Users can browse and filter through a variety of car seats, select the desired rental date and city, add items to the cart, and complete the checkout process using Stripe for payment. Each order is assigned a unique 5-digit confirmation code for easy reference.

Stacks and Technologies Used:

  • MERN (MongoDB, Express, React, Node.js)
  • Stripe
  • Material UI
  • CSS

Sushi-2-Go

Sushi 2 Go is a restaurant ordering application that allows users to browse a menu, add items to their cart, and place an order. The application ensures a seamless communication channel by sending SMS notifications with order details and updates. On the restaurant side, Sushi 2 Go streamlines the order management process, providing an efficient system to receive and confirm incoming orders.

Stacks and Technologies Used:

  • EJS
  • Node.js
  • Express
  • PostgreSQL
  • jQuery
  • Twilio
  • SCSS

TinyApp

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (much like bit.ly). Users can register and login to create, edit, and delete their own shortened URLs. The application also tracks the number of visits to each shortened URL, as well as the number of unique visitors who have visited the URL.

Stacks and Technologies Used:

  • Node.js
  • Express
  • EJS
  • bcryptjs
  • cookie-session

PhotoLabs

The PhotoLabs project is a React-based single-page application that allows users to explore and interact with a vast collection of photos. It leverages a pre-existing API, integrates notifications, and utilizes Create React App (CRA), Webpack, Babel, Express.js, and PostgreSQL for seamless navigation, enhanced user experience, and efficient data management.

Stacks and Technologies Used:

  • React
  • Create React App
  • Webpack
  • Babel
  • Express
